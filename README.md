# 🚗 Vehicle Service Booking System

## 📌 Project Overview
The **Vehicle Service Booking System** is a web-based application designed to simplify and automate the vehicle servicing process. The system enables vehicle owners to book service appointments online, manage vehicle details, and maintain service-related information efficiently.

---

## 🎯 Project Objectives
- Provide a convenient platform for vehicle owners to schedule service appointments.
- Reduce manual service booking procedures.
- Maintain vehicle and service records effectively.
- Improve communication between customers and service providers.

---

## ✨ Key Features
- User Registration & Authentication
- Secure Login and Logout
- Vehicle Information Management
- Online Service Appointment Booking
- Service Information Management
- Booking Management
- Customer Support and Contact Management

## 📂 Project Structure
The system is organized into three main components:

- **Frontend** – User interface development using HTML, CSS, and JavaScript.
- **Backend** – Application logic implemented using Java OOP concepts.
- **Database** – Data storage and management using MySQL.

---

## 🗄️ Database Description
The MySQL database stores and manages:

- User information
- Vehicle details
- Service booking records
- Service-related data

---

## ▶️ How to Run the Project

1. Clone or download the repository.
2. Open the project using NetBeans IDE.
3. Configure the MySQL database connection.
4. Import the required database tables.
5. Run the application.
